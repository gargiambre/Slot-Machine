# Slot-Machine

A JavaScript Slot Machine simulates a classic three-reel machine. It works by taking a amount of money deposited by the player and then taking a user input on the number of lines player want to bet on. Further the bet amount is collected and the slot machine is spinned. The win or loss is checked and the balance amount of the player is calculated. Lastly, player is asked if they want to play the game again or quit.
